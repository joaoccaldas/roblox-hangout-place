# Roblox Hangout Place

An amazing virtual hangout space where users can meet, interact, and enjoy various activities together.

## About

This is a Roblox game designed as a social hangout space. It features multiple themed areas, mini-games, customization options, and a transportation system to move between different environments.

## Features

- **Multiple Themed Areas**: Visit the beach, mountain top, space station, underwater cave, and arcade zone
- **Social Hub**: Central area for meeting and interacting with other players
- **Mini-Games**: Participate in treasure hunts, obstacle courses, and build challenges
- **Customization**: Personalize your avatar with accessories and clothing
- **Transportation System**: Teleport between locations instantly
- **Interactive Objects**: Sit on furniture, use decorations, and more

## Installation

1. Download Roblox Studio from the [official website](https://www.roblox.com/create)
2. Create a new place or open an existing one
3. Copy the scripts from the `src` directory into your place
4. Create the necessary folder structure in Roblox Studio:
   - `ReplicatedStorage/Modules`
   - `ReplicatedStorage/Remotes`
   - `ReplicatedStorage/Assets`
5. Add the RemoteEvents as described in the setup instructions

## Scripts Included

- `main.lua`: Core game functionality
- `social_hub.lua`: Social interaction features
- `activities_zone.lua`: Mini-games and activities
- `transport_system.lua`: Movement between locations
- `customization_system.lua`: Avatar and environment customization
- `model.lua`: World structure definition
- `setup.lua`: Initial configuration
